## Run Locally

Clone the project

```bash
  git clone https://github.com/Vyshnav-KS/nft-marketplace
```

Go to the project directory

```bash
  cd nft-marketplace
```

Install dependencies

```bash
  yarn install
```

Start the hardhat environment

```bash
  npx hardhat node
```

Deploy contracts

```bash
  npx hardhat run scripts/deploy.js --network localhost
```

Start the server

```bash
  yarn run dev
```
